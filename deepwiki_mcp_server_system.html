MCP Server System

Relevant source files
  • lib/llm/prompt.dart
  • lib/mcp/sse/sse_client.dart
  • lib/page/setting/mcp_server.dart
  • lib/provider/mcp_server_provider.dart

The MCP Server System implements the Model Context Protocol (MCP) integration in ChatMCP, providing a comprehensive framework for managing external AI tools and services. This system handles server discovery, configuration, authentication, client communication, and tool execution coordination.

For information about LLM provider integration, see LLM Providers and Clients. For details about tool execution and function calling, see Tool Execution and Function Calling.

System Architecture

The MCP Server System is built around the McpServerProvider singleton that orchestrates all server management operations, multiple client implementations for different communication protocols, and a comprehensive configuration system.
